Independent H-Gates

The two H Gates create qubits with a 50% chance of collapsing to 1. The bits are not correlated.



Measure 0 1
Qubit #0 49% 51%
Qubit #1 51% 49%

Qubit#0 Qubit#1 %
0 0 24%
0 1 25%
1 0 26%
1 1 24%

E Table Qubit #1
Qubit #0 48%

Belle Test

We can entangle the qubits using a cx (controlled not) gate.



Measure 0 1
Qubit #0 52% 48%
Qubit #1 52% 48%

Qubit#0 Qubit#1 %
0 0 52%
0 1 0%
1 0 0%
1 1 48%

E Table Qubit #1
Qubit #0 100%

Qubit #0 is entangled with Qubit #1

Preserving Entanglement

Even if we add new H-Gates to the end of the circuit, the entanglement is preserved.



Measure 0 1
Qubit #0 49% 51%
Qubit #1 49% 51%

Qubit#0 Qubit#1 %
0 0 49%
1 1 51%

E Table Qubit #1
Qubit #0 100%

Qubit #0 is entangled with Qubit #1

Destroying Entanglement

Measuring the qubit before the cnot gate breaks the entanglement.

Qubit #0 49% 51%
Qubit #1 52% 48%
PreMeasure 51% 49%
